Văn Anh prototype UI ở Bolt rồi bàn giao qua file markdown + module thuần đặt ở thư mục **`~/App/song builer/`** (lưu ý gõ thiếu chữ: "builer", có dấu cách). Ví dụ: `SONG_BUILDER_HANDOFF.md`, `tempoFit.ts`, `TapTempoSim.jsx`.

**How to apply:** Khi Văn Anh nói "đọc file bàn giao" / "tích hợp module X" mà không có trong repo `~/Desktop/cs-rhythm-editor`, tìm ở `~/App/` (đặc biệt `~/App/song builer/`) trước khi kết luận không tồn tại.
