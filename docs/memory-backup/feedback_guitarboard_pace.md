Khi chỉnh sửa editor khuông nhạc/TAB ([[project_guitarboard_alphatab]]), làm CHẬM và tỉ mỉ: mỗi lần chỉ xử lý đúng MỘT chi tiết nhỏ user nêu, làm xong cho xem rồi chờ phản hồi.

**Why:** User (thầy Văn Anh) là người soạn nhạc, rất để ý chi tiết nhỏ; "việc chỉnh sửa còn rất nhiều chi tiết nhỏ". Vội vàng / gộp nhiều thay đổi / tự push làm khó kiểm soát.

**How to apply:**
- KHÔNG tự push/deploy trừ khi user yêu cầu rõ ("push", "deploy", "lên web").
- KHÔNG tự thêm cải tiến ngoài phạm vi user nêu.
- Làm từng chi tiết một, chờ user duyệt trước khi sang chi tiết kế.
